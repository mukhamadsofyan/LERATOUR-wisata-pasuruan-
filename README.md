## Tentang LERATOUR

**LERATOUR** adalah aplikasi web pariwisata untuk menampilkan informasi destinasi wisata, kategori wisata, keunggulan destinasi, testimoni pengunjung, serta modul transaksi/pemesanan (invoice) yang dikelola melalui panel admin.

### Fitur Utama
- Landing page modern (template Viatours style)
- List & detail wisata
- Kategori wisata (CRUD)
- Keunggulan wisata (CRUD)
- Slider/hero banner (CRUD)
- Testimoni pengunjung (submit & moderasi admin: pending/accepted/rejected)
- Transaksi/pemesanan tiket & **invoice** (admin dapat cek detail)
- Upload foto (wisata/testimoni/slider) + preview
- Pagination / DataTable (admin)

## Tech Stack
- **Laravel** (Backend + Blade)
- **MySQL** (Database)
- **Bootstrap/Tailwind/Template Viatours** (UI — menyesuaikan project kamu)
